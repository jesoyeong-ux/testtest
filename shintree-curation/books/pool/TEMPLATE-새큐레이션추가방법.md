# 📚 신트리 큐레이션 추가 방법

## 새 큐레이션 추가하는 법

1. 이 폴더(`books/pool/`)에 **`vol-03.json`** 파일을 새로 만드세요.
2. 아래 템플릿을 복사해서 내용을 채우세요.
3. 서버를 재시작하거나 그냥 두면 자동으로 인식됩니다.
4. 앱에서 새로고침(↻) 버튼을 누르면 순서대로 표시됩니다.

## 현재 순서
- vol-01.json → vol-02.json → vol-03.json → ... → vol-01로 돌아옴

## 팁

- **ISBN-13**: 책 뒷면 바코드 아래 13자리 숫자. 네이버 책 검색에서도 확인 가능.
- **표지 이미지**: ISBN이 맞으면 자동으로 네이버에서 불러옵니다.
- **vol 번호**: 파일명이 `vol-03.json`이면 내부 `"vol"`도 `"2026 · Vol.03"` 으로 맞춰주세요.
- **파일명 규칙**: `vol-숫자.json` 형식으로 저장하면 번호 순서로 정렬됩니다.
